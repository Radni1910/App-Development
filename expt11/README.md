## API INTEGRATION 

**Cat Images App (React Native + Expo+ API integration)**

A simple React Native Expo project that fetches and displays random cat pictures using the TheCatAPI.
It shows loading, error, and empty states, supports pull-to-refresh, and displays two cat images per row with a cute banner header.

## Features

- Fetches random cat pictures using https://api.thecatapi.com/v1/images/search?limit

- Displays two images per row.

- Supports reload (pull-to-refresh).

- Handles loading and error states.

- Clean, responsive UI
